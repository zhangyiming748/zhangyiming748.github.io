如何创建一个swapfile

2023-11-07 Linux

** btrfs系统不支持swapfile!!!不支持swapfile!!!不支持swapfile!!!**

sudo dd if=/dev/zero of=/swapfile bs=1M count=<your like size such as 1024>
sudo mkswap /swapfile
sudo swapon /swapfile
