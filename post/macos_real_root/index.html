macOS开启真正的root用户

2022-09-26 macOS

这是原文地址
之所以要盗过来,就是怕这个博文哪天消失了,留个备份,因为这确实是我需要的功能

右键访达选择前往文件夹或者shift+command+G

输入/System/Library/CoreServices/Applications

找到目录实用工具并打开

点击左下角的锁,并打开

点击顶部菜单栏的编辑 启用root用户

给root用户设置密码,设置成功后root用户开启

开启之后就能在用户登录界面的其他用户里选择root登陆了

终端环境默认root权限

在管理员用户下的sudo su sudo -i sudo -s并不是真正的root,即使能做的已经很多了

类比正版Windows系统的administrator账户

没有需求的人尽量别开
