允许当前用户免密使用sudo

2024-08-11 Linux

visudo是好文明

要实现当前用户（uid=1000）免密使用sudo,可以按照以下步骤操作:

  1. 编辑 /etc/sudoers 文件:
sudo visudo
  1. 在文件中找到以下行:
# %wheel  ALL=(ALL)       ALL
  1. 取消注释该行,并在行末添加当前用户的 uid:
%wheel  ALL=(ALL)       ALL  1000

这样就可以让 uid 为 1000 的用户免密使用 sudo 命令了。

如果你想让某个特定用户免密使用 sudo,可以在 /etc/sudoers 文件中添加以下行:

username  ALL=(ALL)       NOPASSWD: ALL

将 username 替换为你想要免密使用 sudo 的用户名即可。
